没有我的团队，我什么也不是

原文：https://medium.com/swlh/i-am-nothing-without-my-team-e1a210f47bab?source=collection_archive---------2-----------------------#2018-06-15

我曾经认为，领导一家公司最困难的部分基本上是了解市场，找出要打造的产品，打造，然后销售和交付产品。我改变主意了。创建一家公司最困难的部分是构建背后的完整“工厂”——人员、流程和文化。特斯拉可能会制造汽车，但事实上，一旦基本创新完成，仍有两个主要问题需要解决:

  • 显而易见的是:建立生产线、供应链、工程、设计、销售和营销
  • 更难的是:团队、内部流程和使其持续成功的潜在文化原则。作为一名领导者，这是一个人每天必须做的工作中更难的部分，“在所有艰难的工作完成之后”。

以下是我在尝试建立团队并成为其中一员时的一些经验，有些团队做得很好，有些团队做得不好——我认为这些东西可以通过示范、重复和结果来学习:

  1. 做你相信的事情:如果一个领导者不仅仅是为了赚钱，而是真正诚实地投身于他们所做的事情，那么相信他、被他激励并追随他会容易得多。作为一名领导者，如果你在做自己不关心的事情，那就很难伪装。
  2. 这不一定是大的传教事业——例如，一些领导人真的只是喜欢解决问题，这表现在他们如何处理问题，分解问题，进入细节，构建和创造选项，然后做出深思熟虑的决定。即使金钱 是 你的产品——比如说作为一名投资银行家或财务顾问——你也可以成为那种将贪婪化身为准备榨干每个客户最后一美元费用的人的领导者，或者成为其他人想要效仿的人，因为你真的证明了你关心如何让你的客户过得更好。同样的工作，作为一个领导者，你的表现却有着巨大的差异。
  3. 建立一种“痴迷足球队”的文化:选择你最喜欢的团队运动，去关注一些比赛，尤其是业余水平的比赛。比如高中足球或排球。 有些足球队无球跑动，排球队“召唤”无球跑动，后退并挑起本应落在另一名队员肩上的重担。没有一句抱怨。为什么？因为他们执着于一件事——竭尽全力去进球，赢得一分，无论目标是什么。我女儿参加过两个排球队——今年的排球队就像是一种积极的文化，他们赢得了比赛，他们真的不应该仅仅依靠技术。去年的球队也是一支技术娴熟的球队，但他们偶尔会相互指责，建立内部派系和领地，“我喜欢和不喜欢的人”，讨好教练。甚至在13岁的时候，他们就表现出了一些同样的潜移默化的组织文化缺陷，他们没有发挥出自己的潜力。要公正。公平点。要透明。变得脆弱。如果你有这种感觉，有时哭一哭。但是要真实。
  4. 作为一家公司招聘不错，但感觉应该像个“椰子”——很难进入，但内部流动。在你确信这是合适的人选之前不要雇佣，然后全力支持你的团队。
  5. 永远学习——这样你就能和你的团队一起解决问题。作为一名领导者，你就像一位家长。当你的孩子犯错时，你可能会在你教给他们的东西、你设定的期望或你的行为模式中分担责任。所以帮他们解决吧。大声演讲会让你表现出虚假的领导力，解决棘手的问题，和你的团队一起度过困难的局面会建立“追随者”。让你的团队成功，他们就会信任你。不管我的头衔如何，如果我不不断学习，在不久的将来，我将无法帮助解决实际问题，因为我离现实太远了。对任何领导人来说，这都是一个危险的局面。
  6. 发扬优势，挑战你的团队，不要只是评判他们:
  • 雇佣那些有好奇心、喜欢成长和热爱挑战的人。给他们一份只要他们努力就有可能成功的工作。但是根据定义，挑战是艰难的。因此，当他们失败时，就像我们都做的那样，与他们一起工作，启发他们，激励他们或教他们——无论他们在“技能-意志”谱上处于什么位置。
  • 给你的团队积极坦诚的建设性反馈。但是不要不停的评价他们。他们在正式评估中看到的或听到的都不应该是真正的惊喜，因为你已经花了时间和精力来创造对他们有意义的角色，并给他们时间和动力，如果他们努力的话，他们就会成功

7 .。激励、指出、委派，但不要放弃:正如这个人所说，不要雇佣优秀的人，告诉他们需要做什么的最后细节——如果你确实为正确的工作雇佣了正确的人，并且如果你倾听，他们经常会给你惊喜。有时候他们会告诉你一个更好的方法去做需要做的事情。所以，倾听，询问，让他们告诉你他们的计划，给他们指出其他的资源，人和专家，然后让他们放松。但偶尔参加正确的深入对话，旨在解决他们面临的问题，并确保他们步入正轨。因为责任的委派与放弃截然不同——归根结底，你和你的团队成员一样负有责任

8。公平地、敞开心扉地给予奖励:如果你不是真心的，就不要说“干得好”。并用事实和相应的奖励结构来支持它。不要羞于坦率地告诉你的团队需要改进的地方。

9。公道正派，不只是好听:我不把我的团队叫做我的家人。原因是在我家如果我是那个没人喜欢的表亲或者叔叔，他们至少在感恩节还是要忍受我。在我的专业团队里，如果我不生产，不交付，不适合，我就不属于。你有时会犯招聘错误。富有同情心有时意味着放手——这对每个人来说都是最好的。不要轻易去做，但是如果这是正确的事情——出于表演或文化原因，就让他们去做，快速地做，带着尊严和同情心去做。

10。做一名球员教练——边做边教:我有过“告诉我该做什么”的老板，也有过“至少一次示范给我看如何做”的老板。例如，为你关心的行为树立榜样，比如“永不放弃”。通过展示你的“蹦床指数”——你从高处跳下的速度是你未来成功的最大潜在指标，来教导你的领导和团队这一点

最重要的是，永远记住，没有你的团队，你什么也不是。
