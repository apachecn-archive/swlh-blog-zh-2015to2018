# 凭直觉写你的小说

> 原文：<https://medium.com/swlh/write-your-novel-intuitively-a27ee6d6701>

![](img/908502c6f989b982fa7d15642b1f5dfa.png)

最近，当我帮助一位同事写小说时，我意识到我想帮助和支持更多的作家。我首先想到的是分享我写第一部小说的方式。

我凭直觉写的。

简而言之，我从一个*假设*的情况开始，让它自己发展。
