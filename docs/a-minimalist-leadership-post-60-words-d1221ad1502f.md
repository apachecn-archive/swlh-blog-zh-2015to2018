# 最低限度的领导职位(60 字)

> 原文：<https://medium.com/swlh/a-minimalist-leadership-post-60-words-d1221ad1502f>

![](img/47ec0ca8a15ecb46461ddae481835590.png)

## 我建了一个小小的领导之家。进来看看吧。

**是**

勇敢。

诚实。

富有同情心。

正宗的。

有创意。

弹回的

承诺。

故意的。

留心。

包容。

好奇。

**做**

关心。

问题。

连接。

感激。

听着。

澄清。

挑战。

风险。

失败。

激发灵感。

鼓励。

**选择**

去爱。

去服务。

去信任。

来分享。

去学习。

成长。

想象一下。

开始吧。

来建造。

去战斗。

完成。

> 一颗橡子可以创造一千片森林—拉尔夫·瓦尔多·爱默生
