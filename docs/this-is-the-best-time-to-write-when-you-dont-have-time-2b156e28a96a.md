# 当你没有时间的时候，这是写作的最佳时机

大学毕业后，我不再担心找不到时间写作。作为一名新闻记者，我必须一直写作。写作必须要快，因为最后期限不能等。有时我没有时间重写；初稿是唯一的草稿。

一场争论不休的镇董事会会议持续到晚上 11 点后，我会回家了解这个故事…
