# 这就是我们要待的地方。

> 原文：<https://medium.com/swlh/this-is-the-space-to-be-in-59f7d0ae2e97>

这里比以往任何时候都需要你的伟大。

![](img/e926435eecc83e2514a5dc5cde324899.png)

亲爱的企业家:

你有天赋，有动力，渴望有所作为。

如果有，请继续阅读。

有三种力量*将*拯救世界，然后*才能*杀死它。

1.  贸易
2.  技术
3.  时间

作为人类，我们正逐步正确对待前两者。越来越多的资源被部署在那里。因此，希望我们能在这些行为奴役我们之前掌握它们。

世界正在学习无边界的贸易(T4 ),尽管有自己的一系列问题，但仍在大步前进。这阻止了他们去打仗。它带来的价值创造和个人自由将在即将到来的时代被证明是不可估量的重要。

长期以来，我们一直(过度地)试图驯服我们蓝点上的资源。更多的**技术**和像蓝色起源/SpaceX 这样的倡议有望驯服其他岩石，这样我们就可以在将资源开采的压力转移到其他地方的同时，保留我们可爱的家园。

正是我对第三个问题的思考促使我给你写信。我称之为缺失的第三个向量——**时间**。我不会谈论太阳会烧尽，地球会爆炸，宇宙会崩溃的时间。我知道我们在这里的时间只是无数其他世界中进化的一秒钟的一小部分，在这一小部分中，我们必须努力继续向前迈进。正是本着这种精神，我谈到我们需要时间来改变我们与现实的关系。我们的物理老师会要求我们找出平衡问题中两种力的“第三种”力。我终于破解了！不仅仅是第三种力量平衡了其他两种力量，如果没有这种力量，其他的解决方案看起来就像是临时的解决方案。因为，无论我们做什么，如果我们不解决根本问题，我们也会找到摧毁它的方法。举例来说——闭上眼睛，想象你最喜欢的未来科幻系列。我的是广阔的，在几代人的时间框架内，当我们开采小行星，有星际维和机构，新的文化规范(如养育)——唯一不变的是我们对彼此如此刻薄的能力。

任何改变的希望都取决于 1)变成另一个物种；2)在我们与我们的世界/现实/模拟的关系中的航向修正

虽然第一个是随着时间或技术的自然选择，但作为你自己，你会选择第二个，就像你选择前两个行动一样。

那么，我们如何正确的航向呢？

**心智升级。**

这样措辞的话，这是科幻电影中又一个涉及政府机构的邪恶阴谋。

但是，冥想/正念/觉知只不过是一种精神控制和精神升级的形式。

观察者就是创造现实的东西(双分裂实验)。那么，为什么我们不在每一个层面上观察我们自己的现实，以确保正确的创造呢？它不是新的，但它是一个开创性的工具。很多人在做的时候甚至没有意识到他们在做。不幸的是，很多应该直接去做的人没有去做！我们怎么能把这样的事情留给机会，作为终身学习的副产品呢！许多应用程序、文学作品和一波复兴浪潮理所当然地开始了，以回应我们生活的时代。

当务之急是使之系统化。为了赢得与时间的赛跑，解决方案是像我们在其他生存挑战上所做的那样，传递我们所有的力量。我们正处于及时干预的临界点。(任何数据科学团队都会告诉您这个“空间”已经成为投资热点等等)。正如这句名言所说(出自达赖喇嘛之手)，如果今天所有八岁的孩子都学会某种形式的冥想，下一代就不会有战争。这些地区、国家乃至最终整个世界都会稳定下来。它只能意味着一件事——更多的繁荣、价值创造和福祉。快速的谷歌搜索将会带来许多科学(同行评审的可复制的)研究，这些研究总结了这种做法在个人和地区层面上的好处。

我们幸运地生活在好时代的一个标志是，身体上的问题开始消失，带来精神上的问题。[世界上死于自杀的人数比所有人类暴力行为的总和还要多，包括恐怖主义](https://ourworldindata.org/what-does-the-world-die-from)。不幸的是，这仅仅是一个开始。随着我们进入普遍自主和分散技术的人类世时代，提升我们的精神比提升我们的身体变得更加重要。

冥想是一条给予的河流。[这当然与我和我直接认识的人有关](/swlh/whats-the-purpose-of-life-ff9d7e07e4b9)。很难用我们所知的语言来表达。

我双手合十，极其谦卑地真诚希望，你们将全力支持这一事业，因为这一次将需要另一种佛来改变这一进程！

问候，

诗缇
