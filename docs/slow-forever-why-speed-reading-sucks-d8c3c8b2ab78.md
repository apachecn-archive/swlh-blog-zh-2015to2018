# 什么是速读？

我打赌你喜欢阅读。或许，像我一样，你也是一个喜欢高效完成工作的书呆子。我在这里要说的是，要不惜一切代价避免的一种“生产力”形式是快速阅读。

快速阅读非常符合在更短的时间内做更多事情的生活习惯。你是一个抓住生活的实干家吗？
