我们重新思考我们的办公室

我坐在第11区的Le Pure咖啡馆外面，开着笔记本电脑，啜饮着寒冷下午的第三杯浓缩咖啡，周围都是游客和当地人。现在还是纽约的清晨，当我修改一份共享的谷歌文档时，来自 Code Morris 团队的消息开始出现在Slack上。今天晚些时候，我们将使用UberConference和Worklife召开一个跨越四个时区的会议。

我正在巴黎进行为期一个月的“工作旅行”，这件事是因为 NodeSource 发生的。

去职还是不去职

当我第一次见到NodeSource的首席执行官兼联合创始人Joe McCann时，他解释说该公司是“分布式设计”，这意味着他们没有中央办公室或总部。相反，员工可以在他们选择的任何地方工作，在世界的任何地方。“无办公室化”是乔和联合创始人丹·肖(Dan Shaw)热衷的一个想法，这很有意义，因为它类似于Node中开发的应用程序的模块化本质。

使用NodeSource让我们直接看到了这种分布式架构的灵活性。我们的团队完全在云中协作，NodeSource团队的成员来自世界各地——家庭、机场、咖啡馆，甚至海滩。

这些互动最终启发我们重新思考我们在Code Morris的工作环境。在一个共同工作的空间里，我们已经被松散地束缚住了。安全、网络、公用设施，甚至咖啡都是为我们管理的。我对办公室最大的贡献是我自己安装在墙上的衣帽钩。我们问自己是否应该放弃我们的办公室，也去完全分布式。我们可以省去通勤，同时还能省钱。这个想法很有说服力。

我们解构了我们的流程，弄清楚我们工作的哪些部分是特定于地点的，哪些不是，哪些交流需要实时，哪些可以异步。事实证明，我们的工作比我们最初设想的要灵活得多。

这种新发现的自由部分是由于NodeSource去年向我们介绍的Slack。几周之内，Slack变得不可或缺。它不仅取代了GChat、短信和许多笨拙的电子邮件，还取代了许多办公室里的闲聊。

我们意识到我们不需要离得很近才能一起工作。物理上的接近成为一种选择，而不是一种要求。

最终，我们决定保留我们的办公室，但要有一个全新的视角。放弃工作是关于同一栋建筑中的一个人的想法，让我们以更少的假设和更多的意图来看待我们的空间。这就像发现了一项我们从未意识到的公司福利。

夏天开始的时候，我们尝试着稍微散开一点。

我们的首席执行官Andy Morris ，开始偶尔在他位于康涅狄格州的海滨别墅里工作。我经常在布鲁克林的沙发上或咖啡馆里工作。我们仍然继续出现在办公室，但远程工作被证明是一种非常无缝的体验。我们的高级客户总监凯蒂爱上了洛杉矶，并决定永久搬到那里。现在，她在西海岸的一个联合办公空间里有一张办公桌，一切都没有太大变化。

所以当我提到要和我的女朋友一起去巴黎住一个月时，安迪的反应是“你当然应该去。”

几周后，我做到了。

移动盛宴击败免费午餐

欧内斯特·海明威将在巴黎的生活描述为“一场流动的盛宴”——这种体验将伴随你的余生。这种观点在今天至少和他在近一个世纪前写下它时一样正确。不同的是，现在你不需要成为驻外记者或小说家就可以在任何地方工作。互联网使得无限数量的移动盛宴成为可能。

巴黎是一个工作的好地方。无处不在的咖啡露台和温和的气候非常适合在夜晚逗留。对于不太喜欢早起的人来说，在纽约时间提前六个小时是非常好的。

旅行工具

准备在海外工作几乎不需要什么计划。我的大部分准备工作包括研究无线数据选项。我最不希望的事情就是生活在国外，却因为连通性而被困在家里。

这些工具让在海外工作和生活成为可能:

  • MacBook Air和iPhone — 这些看似显而易见，但却值得一提。鉴于我上一次去巴黎需要带一台摄像机、一台单反相机、各种镜头、电池、一台沉重的笔记本电脑、备用电池、各种充电器以及无数的电缆、连接器和适配器，我惊讶于这次我带的东西如此之少。我的iPhone不仅取代了所有的摄像设备，还取代了纸质书籍和地图。
  • Slack——我在上面提到过这一点，但是Slack提供的持续部分连接的水平创造了一种真正的“在工作”的感觉，而不会感到打扰。
  • 谷歌应用套件(Google Apps)——尽管Slack提供了异步通信的所有好处，但在实时文档上进行协作还是有一些私密性的，即使那些其他光标是由遥远的键盘驱动的。谷歌文档让我有生之年再也不想打开另一个有修订的Word附件。
  • 我们已经改用UberConference进行电话会议了，但是因为我停用了这个月的电话号码，所以需要有网络连接的能力。当您绝对需要拨打座机时，Skype点数也是一个不错的备用选择。Google Hangouts也提供了类似的功能。
  • 便携式4G WiFi热点— 从hippocket WiFi租赁一个便携式WiFi热点被证明是巴黎互联网连接的最佳选择，因为公共WiFi通常速度慢且不可靠。即使是解锁的手机，预付费SIM卡选项的激活和充电也很麻烦，很难区分哪些计划和提供商启用了移动热点功能来连接笔记本电脑。最后，对于两个带着手机和笔记本电脑的人来说，租一个便携式WiFi热点是最好的选择。这个月我们通过禁用美国无线账户省下的钱基本上支付了租金——15GB 4G数据大约150美元。这意味着没有短信或传统的语音通话，但老实说，我没有注意到。iMessage、FaceTime、Hangouts和Skype让我在这里保持联系，就像在纽约一样。
  • 信用卡——我最终得到了一张支持芯片和密码的巴克莱信用卡，没有国外交易费。芯片和密码卡在欧洲不是必需的，但是它们在只刷美国卡的地方可以用(比如自助地铁站)。

这份清单绝非详尽无遗。语言障碍可能具有挑战性。管理从实体邮件到ATM费用的一切变得更加复杂。远离家乡的医疗保健和药物治疗很棘手。尽管像AirBnB和优步这样的公司可以让数字流浪者的生活方式感觉没有摩擦，但机场很乐意提醒你事实并非如此。

但与原地不动相比，这种权衡是值得的。

去而复返

将我们的办公室重新想象为一种选择而不是一种给定，开启了我们未曾考虑过的新的可能性:一边工作一边探索世界，在新的地方生活而不是将旅行压缩成短暂的假期，定义一种生活方式而不是由工作定义的生活方式。仅仅是选择住在一个不太昂贵的城市就是一个有价值的好处。

与其竞相提供免费午餐和桌上足球等额外福利，公司可能会更关注生活方式的好处。研究表明，最终让人们更快乐的是T2的经历，而不是事情。你不需要被束缚在一张桌子上、一间办公室里，甚至一个国家里才能富有成效。

即使我最终不会连续出差，保持与物理位置更松散的联系，减少财产，保持更大的流动性，以备下一次在任何地方工作的机会到来时，仍然有价值。
