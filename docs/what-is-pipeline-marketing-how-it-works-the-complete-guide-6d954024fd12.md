有许多不同类型的营销:内容营销、内向营销、外向营销——不胜枚举。管道营销可能看起来像另一个时髦词，但它实际上植根于可靠的数据。

作为一名企业家，你可能对你的[在线课程](https://blog.newkajabi.com/mega-guide-to-creating-an-online-course-part-1)有很多目标。你想分享你的教育，结识新朋友，创造漂亮的产品，并在你的行业中树立权威。

然而，有一个目标可能是压倒一切的:你想赚钱。

管道营销为企业家提供了一个解决方案，这些企业家在创造销售线索方面没有困难，但却很难将销售线索转化为客户。

潜在客户是您收集了其联系信息的人。仅此而已。销售线索是潜在客户。

另一方面，一个顾客却用他或她的血汗钱来换取你的一个数码产品。显然，客户比销售线索提供更多的价值。

为了让你的[在线课程](https://blog.newkajabi.com/have-you-checked-out-kajabi-university-yet)走上成功的快车道，你可能想尝试一下管道营销。

请继续阅读，了解管道营销的优点，并了解它与简单的销售线索挖掘有何不同。你可能已经知道如何产生销售线索。现在你需要想出如何最大化销售。

# 什么是管道营销？

![](img/5d56f20353d3e0a0f039adb70885dfdc.png)

管道营销将销售和营销数据结合起来，专注于将潜在客户转化为客户，而不是继续创造更多的潜在客户。企业家可以利用管道营销使他们的销售过程更有效率，成本更低。

将管道营销视为将您的销售和营销努力汇聚到最重要的地方的一种方式。这是一种策略，可以让你专注于营销漏斗中较窄的部分，而不是顶部。

# 漏斗顶部:产生销售线索

在漏斗的这一点上，你只是试图接触尽可能多的人。这就像在海洋中撒下一张大网，尽可能多地捞起你能捕获的鱼。

你可以通过多种方式收集线索，比如打陌生电话，邀请网站访问者注册你的电子邮件列表，以及在社交媒体上与潜在客户联系。

漏斗顶部旨在获取尽可能多的潜在客户信息。它通常被称为“意识”阶段，因为这是你让潜在客户意识到你的品牌的时候。

例如，如果你在[提供免费下载](https://blog.newkajabi.com/how-to-collect-emails-like-crazy)后收集电子邮件地址，你可能会通过[的滴滴电子邮件活动](https://blog.newkajabi.com/kajabi-drip-simple-email-automation-for-your-kajabi-sites)跟进订户。你将解释你所提供的内容，详述你通过在线课程所创造的价值，并为那些订阅者购买你的数字产品提供令人信服的理由。

然而，你没有办法知道为什么那些人首先接触到你的业务。

# 漏斗中部:培育潜在客户

当你移动到漏斗的中间时，你开始培养销售线索，鼓励他们成为客户。你可以邀请他们参加免费的网上研讨会，向他们展示你的课程预览，或者提供促销折扣或优惠券。

# 漏斗底部:将销售线索转化为客户

漏斗的底部是潜在客户成为客户的时候。他们实际上交出他们的信用卡来换取你的数码产品。

不幸的是，许多企业对漏斗的中间和底部不够重视。相反，他们继续寻找线索，希望其中一些人会改变主意。

这是一种适用于许多企业的策略。只是效率不是很高。

这些数字表明，瞄准销售渠道的不同部分可以对你的实际销售产生更深远的影响。

销售渠道是什么？

销售渠道是你的客户从第一次接触(当他们第一次接触你的品牌)到第一次购买的过程。随着你的客户对你的企业越来越忠诚，这种情况会持续到以后的购买中。

# 管道营销的力量

![](img/b7cd6ed075ebd50aad6e21b8296acf93.png)

管道营销的力量在于它能够将营销和销售统一在一个屋檐下。这对拥有小团队的创业者尤其有效。如果你同时担任营销人员和专业销售人员，你必须将这两个活动视为一个整体。

从本质上来说，管道营销可以让你直接将营销工作与未来收入联系起来。你不需要使用市场营销来产生潜在客户和销售，而是将它作为同一个过程的一部分。

虽然这听起来令人困惑，但一旦你理解了背后的心理和力量，就很容易实现。

首先，让我们从 [HubSpot 关于销售线索转化统计](https://www.hubspot.com/marketing-statistics)的一些确凿事实开始:

*   大约 80%的营销人员认为他们的潜在客户创造策略只是有些效果。
*   超过 50%的营销人员将其总营销预算的至少一半投入到潜在客户开发中。
*   几乎四分之一的营销人员不知道他们的潜在客户创造努力是否真的会导致成交。
*   培养销售线索可以增加 20%以上的销售额。

考虑到这些事实，营销人员如何利用管道营销来提高总收入？

从本质上讲，这种营销策略帮助企业家集中精力培养合格的销售线索，而不是简单地抓住尽可能多的线索。

这听起来可能违反直觉——毕竟，营销人员总是在谈论品牌影响力和线索生成——但线索没有价值，除非那个人真的购买了产品。

# 销售线索挖掘和管道营销的区别

![](img/8ff378c868857d30902f4a11fc0b6644.png)

线索挖掘和渠道营销是实现同一目标的两种不同方法。潜在客户的产生完全取决于数量——尽可能多地获取电子邮件地址和其他形式的联系信息。

另一方面，管道营销就是从已经存在的潜在客户那里创造收入。换句话说，企业家更注重将销售线索推向销售。

听起来很合理，对吗？毕竟，不管你有多少线索，它们对你的业务没有任何实际价值。

因此，让我们来看看，随着你销售更多的在线课程，管道营销和营销收入如何为你带来优势。

# 它是如何工作的

管道营销活动始于一系列具体目标。一旦你知道你想要达到的目标，你就找到了将营销和销售活动结合起来的最佳方式，并将你的战略付诸实施。

您关注的是每个机会的成本，而不是潜在客户的产生。每次机会的成本越低，你的现金流就越大。然后，你继续测量和监控[重要指标](https://blog.newkajabi.com/introducing-kajabi-video-stats)——那些导致销售的指标——并继续完善你的技术，直到你掌握管道营销。

但是，让我们稍微分解一下，以便您了解如何将管道营销付诸实施。

# 定义活动目标

除非你有明确的、可衡量的目标，否则任何营销计划都不值得追求。你想通过你的营销努力达到什么目的？

“赚更多的钱”不是一个具体的或可衡量的目标。可以说，这是任何企业家的愿望，但它不会帮助你创建一个成功的营销策略。

从检查你过去的销售记录开始。在过去的六个月中，您每月销售多少在线课程？与你创造的潜在客户数量相比，你完成了多少销售？

了解这些数据点可以帮助你设定一个合理的目标。

例如，假设在过去的六个月中，您每个月都创造了 2000 条销售线索。在这 2000 条线索中，有 1000 条购买了您的产品。换句话说，你已经将 50%的潜在客户转化为付费客户。

这些数字真的很好。但是，你可以让他们变得更好。

您可能会发现，您只成功转化了总销售线索的 1%到 2%。那也没关系。这是设计糟糕的营销策略的一个症状，但是管道营销可以帮助你提高你的数字。

你可能决定让你的营销活动将你的转化率提高到 5%。当你从 1%或 2%上升时，这是一个合理的目标——至少在短期内是如此。在你建立一个更高的目标之前，你想测试不同的策略，并确保它们有效。

你也可以给你的目标添加更难的数字，比如:

*   在接下来的八周里，每周销售 X 门在线课程。
*   将营销和销售计划的投资回报率提高 X%。
*   在接下来的两周内产生 X 个新客户。

当然，这个数字取决于你的具体业务和你的历史。

如果你还没有卖出任何在线课程怎么办？也许你刚刚开始经营你的生意。

那也没关系。在这种情况下，根据竞争对手的成功建立合理的目标。进行一些竞争对手的研究，以确定你的受众有多少，他们对你想要创建的在线课程类型有多渴望。

# 确定实现目标的最佳方式

现在你知道了你想要达到的目标，你需要采取行动来实现它。在管道营销中，你不会太担心漏斗的顶端。继续收集电子邮件地址，参与社交媒体，结识新朋友，但是把更多的注意力放在漏斗的底部。

看看你已经产生的潜在客户，根据你对他们过去行为的了解，开始对他们进行分类。他们是否不止一次与你的业务有过互动？他们给你发邮件询问更多信息了吗？他们是否出席或参与与您的业务相关的活动？

根据这些数据，您可以开始确认您的销售线索。换句话说，你只想把时间花在可能真正转化为客户的人身上。

调查对于确定销售线索是非常宝贵的，尤其是如果你没有全职销售人员随时待命的话。

发送一份调查问卷，询问旨在确定他们在购买者购买过程中的地位的问题:

*   您是否计划在未来六周内购买[您的主题]课程？
*   你学习[主题]的目标是什么？
*   具体来说，我的课程能帮助你在个人或职业生活中实现什么？

提出具有开放式答案的问题。这样，你的潜在客户会与你分享更多信息。

根据调查答案，你会对那些似乎准备转变的潜在客户有所了解。然后你可以在这个过程中花更多的时间来培养他们。

# 关注每个机会的成本

管道营销是一种基于结果的营销系统。它旨在帮助您实现创收的特定目标。

每机会成本是比每线索成本更有效的指标。它在你的营销(不是销售)努力和最终结果之间建立了联系。

换句话说，你的营销努力对你从在线课程销售中获得的收入有影响吗？如果没有，你如何调整你的营销策略，使他们更有效和基于收入？

在大多数管道营销活动中，当客户转化时，每项营销努力都会得到重视。让我们看看一个虚构客户的快速时间表。

客户搜索与您的在线课程相关的主题，并找到您就该主题撰写的博客文章。客户注册您的电子邮件列表，并下载您宣传的免费报告。几周后，客户点击其中一封电子邮件中的链接，进入你的销售页面。在那里，他或她选择课程并购买。

在这种情况下，几项营销努力促成了最终的销售:

*   有机搜索和内容营销
*   基于激励的[电子邮件营销](https://blog.newkajabi.com/you-ve-got-mail)
*   电子邮件点击率
*   销售页面

当分析你的营销指标时，你会将来自该客户的收入的 25%分配给上述每一种营销策略。每个人都有 25%的责任来转化销售线索。

你对每个顾客都这样。最终，你会发现哪些营销努力确实有助于销售，哪些没有。你可以消除无效的方法，这样你就不会在不合格的线索上浪费精力。

您的机会是最终进入销售渠道的销售线索。他们继续与你的业务互动，并通过管道最终购买。不要计算每条线索的成本(意思是你收集的每条线索的成本)，而要关注每一个机会的成本(获得每条进入渠道的线索的成本)。

# 监控和测量

![](img/96357a1e348e2bda5df924274782af23.png)

每个营销和销售策略都依赖于指标，但如果你想产生结果，管道营销可能是最依赖大数据的。

但是，我们的目标不是增加您跟踪的指标数量，而是选择对实际收入和销售影响最大的指标。

在管道营销中，大多数企业家跟踪以下指标:

*   按渠道划分的收入:你从每个营销渠道(如电子邮件、社交、点击付费和其他)获得多少收入？
*   销售线索量:在特定时间段内，您收集了多少销售线索？
*   关闭的客户:有多少客户真正从你的网上商店购买产品？
*   营销投资回报率:你的营销策略产生了多少实际收入？
*   销售线索来源的收入:你从每个销售线索来源获得了多少收入？
*   按渠道划分的机会:对于您的每个营销渠道，有多少线索变成了机会？

本质上，您应该跟踪销售团队在关闭客户时跟踪的相同指标。你正在缩小营销和销售之间的差距，以确保你的重点仍然是收入而不是线索。

# 调整，收获回报

随着时间的推移，您将跟踪前面提到的指标。随着你从努力中获得更多的数据，你可以改进你的策略。

例如，也许你没有从社交媒体营销中创造很多机会。在这种情况下，你应该减少社交媒体的投入，将更多精力投入到能带来实际收入的渠道中。

随着时间的推移，你会发现你的收入增加了，而营销和广告支出却减少了。这就是成功的定义。

# 销售线索挖掘与管道营销

我们已经讨论了销售线索挖掘和管道营销，但是它们到底有什么不同呢？显然，任何类型的营销要想成功，你都需要线索。然而，销售渠道证明，你不需要过多的线索来建立一个创收业务。

让我们来看看销售线索挖掘和管道营销之间的一些具体区别，以便您可以成功地完善您的营销策略。

# 漏斗顶端焦点与全漏斗焦点

如上所述，销售线索挖掘侧重于漏斗顶部。它旨在将尽可能多的潜在线索注入漏斗。

有点像头脑风暴。你知道你想要创造一些东西，所以你拿着一张纸和一支笔坐下来。你让你的想象力天马行空，记下你脑海中出现的每一个想法。

最终，到了真正创造东西的时候，你不会用到很多这样的想法。其中一些可能毫无意义，而另一些则不够具体、有趣、吸引人或宽泛，不足以保证实现。

头脑风暴是一个很好的练习，但是它也会产生很多没用的想法。你把这些想法放进你的精神垃圾桶，只带着那些可能会变成你有用的东西的想法前进。

销售线索生成也遵循同样的原则。企业知道，无论他们收集了多少线索，有些永远不会变成客户。尽管如此，他们还是不断向漏斗中注入线索，希望至少有一些人会购买他们的产品。

管道营销则不同。它没有切断漏斗的顶端——如果它切断了，一个企业家怎么能产生任何销售呢？—但这需要一个完整的漏斗方法。换句话说，在捕捉到线索后，营销和销售团队努力让尽可能多的人通过整个漏斗，直到他们最终成为客户。

让我们暂时回到头脑风暴类比。

如果不是写下你脑海中出现的每一个想法，而是花几秒钟仔细考虑每一个想法，会怎么样？在那段时间里，你也许可以消除其中的大部分。在头脑风暴会议结束时，你的想法会更少，但每个想法的质量会更高。

这就是管道营销的目标。即使你产生的潜在客户较少，你也可以依靠这些潜在客户成为客户，因为你确切地知道如何引导他们通过销售渠道。

# 慢速销售与快速销售

当您关注较少的销售线索时，您可以更快地推动销售线索通过漏斗。这自然会导致更快的销售。

想想这些可能性。假设你有 10 条线索变成了机会。他们在管道中，准备好让你在剩下的过程中培养他们。

相比之下，你有 50 条线索。你不可能对 50 个人投入和对 10 个人一样多的时间和注意力。

更快的销售意味着更快的增长。随着您的在线课程越来越受欢迎，您将有更多的时间来创建更多的课程，专注于客户忠诚度，并推动进一步的收入。这是一个循环，从管道开始，但永远不会停止。

更快的销售也意味着您的销售线索有更少的时间说服自己放弃购买。你可以向他们展示参加你的课程会给他们带来的好处，然后让他们做出决定。如果你足够有说服力，更多的线索会成为客户。此外，你的潜在客户可能会告诉他们的朋友，他们会自动成为更合格的潜在客户，因为他们是通过推荐而不是互联网搜索或 PPC 广告找到你的。

# 创建联系人与创建客户

人脉很有价值。当你可以通过电子邮件或社交媒体与人们取得联系时，你更有可能将他们转化为客户，而不是完全没有联系方式。

然而，不要搞错:电子邮件地址不是一笔买卖。这只是出售的可能性。

企业家还在经营，因为他们能够创造收入。没有现金流，你无法继续创建在线课程，因为你需要找到另一种方式来提供食物。

因此，把 90%的精力放在建立联系上是没有意义的。你会得到更多的线索，但你可能一个客户都没有。

潜在客户开发侧重于收集联系人。管道营销侧重于创造新客户。后一个目标对您的企业和客户都更健康。你不会浪费时间在那些从来不想转化的潜在客户身上。

# 排除与包含转化营销

转化营销是指鼓励销售线索、潜在客户或机会采取下一步行动的营销策略——无论是什么样的行动。

线索生成通常不包括转化营销。它关注的是通过漏斗注入尽可能多的联系人，所以它没有考虑到让人们转化的需要。

想想自己的事。你希望人们购买你的课程，但是潜在客户和潜在客户可能需要在第一次接触你的企业和实际购买课程之间采取一些步骤。

他们可能会注册您的电子邮件列表(一项行动)，点击您电子邮件中的链接阅读您的博客文章(行动)，并观看您录制的一些网络研讨会(更多行动)。这些步骤中的每一步都代表一次转换。

但是，最终转化的还是购买。那是你应该集中大部分精力的地方。

# 虚荣指标与收入指标

虚荣指标让我们感觉良好。举例来说，如果一篇博客帖子像病毒一样传播，并在一天内吸引了 10 万人访问你的网站，你很可能会打开最近的一瓶香槟。你对成功的机会和潜力感到兴奋。

然而，就像线索一样，网站访问者不会把钱放进你的口袋。web 流量、电子邮件打开率和免费下载等指标可能会让你觉得自己是营销明星，但它们对销售没有任何直接影响。

当你只关注或主要关注潜在客户生成时，你可能会被虚荣心指标引入歧途。你会开始认为自己在商业上取得了成功，但实际上，你只是在增加流量、写诱人的电子邮件或创造引人注目的免费下载方面取得了成功。

另一方面，收入指标是与收入直接相关的指标。如果潜在客户下载了免费赠品，然后立即购买了您的课程，您可以将这些指标作为创收流程的一部分进行跟踪。

然而，大多数收入指标都是基于销售漏斗的底部。它们针对的是成为机会并进入销售渠道的潜在客户。

例如，收入增长告诉你在一段时间内你的收入增长了多少。如果你的收入稳步增长，你知道你的管道营销战略正朝着正确的方向前进。

# 效率较低的预算与效率较高的预算

![](img/d35361e74494634acf135f185ebc4e1b.png)

现在你更熟悉管道营销是如何运作的，你可能会看到你将更有效地使用你的营销预算。

如果您在合格的销售线索和机会后投入更多资金，您将获得更多收入，并且您的投资回报也会随之增加。潜在客户的产生经常把钱冲进马桶，因为你把钱投入到一个并不总是能产生付费客户的策略中。

# 卡贾比管道

你如何将管道营销应用到 Kajabi 的在线课程中？

*   关注关键指标:选择与收入而非销售线索相关的关键绩效指标(KPI)。
*   培养合格的潜在客户:与其追逐不断增长的稳定潜在客户，不如专注于培养你已经开发的潜在客户。注意他们的行为接触点，这样你就知道什么时候和他们接触。
*   避免虚荣心:不要太相信网站流量和邮件打开率。相反，关注推动收入的指标。
*   把营销和销售联系起来:把营销和销售想象成一台大机器。观察导致实际销售的模式，而不仅仅是更多的接触点。
*   跟踪你的投资回报率:密切关注那些不能转化为金钱的营销活动。如果它们过时了，就把它们清除掉。

# 结论

管道营销似乎有点违反直觉。毕竟，我们被教导要让尽可能多的人关注我们的内容和品牌。

然而，潜在客户并不是你的收入来源。你不能把线索拿到银行去兑现。领导也不能把食物放在桌子上。

相反，最成功的企业专注于基于收入的转换、指标和营销。他们希望有尽可能多的机会转化为客户。
