# 给初次创业者的最佳建议

> 原文：<https://medium.com/swlh/best-tips-for-first-time-entrepreneurs-34760bb86ed6>

![](img/23da687bf3b6a4748902ed36554629dc.png)

Top 19 Tips For First Time Entrepreneurs

纵观全球，从东欧、尼日利亚、斯堪的纳维亚到印度，我们见证了一代人受到创业热潮的影响，这些地方远不止著名的硅谷。这一代企业家将改变世界经济。希望你能在变得更好的过程中发挥作用。再多的阅读也不会让你完全为未来的旅程做好准备，因为每个人都有一条独特的道路，这就是创业之旅如此激动人心的原因。即便如此，这份清单应该会让你对这个旅程有所了解。

# 1.你的商业理念应该会自我推销

意思是，它应该很容易在当前的市场上找到一个位置。弄清楚你的服务或产品是否有很大的市场。投资者通常只有在看到一家公司有发展壮大的潜力，或者注意到一个重大的市场机会时，才会投资这家公司。

# 2.激情是会传染的

为一家大公司工作意味着公司的品牌和声誉会为你做好准备。你可以获得活动的免费门票，代表们为你提供美酒佳肴。世界似乎围绕着你转。当你开始创业的时候，派对就结束了。你只能靠自己了。从这一点上来说，你必须赢得人们的注意。你可以通过超越你的竞争对手来做到这一点。比别人多关心一点。激情是你最宝贵的财富，尤其是在开始的时候。你的能量会让人们相信你和你的项目，尽管还没有什么值得相信的。

# 3.网络将被证明是救命稻草

它可以让你成为一名优秀的员工，一名新的投资者，一名新的客户，或者一名优秀的导师。参加创业和行业活动。LinkedIn 是帮助你建立人际关系网的强大工具。确保你和你的公司都在 LinkedIn 上有个人资料，并确保定期添加相关的新关系。当有人在谷歌上搜索你时，你的 LinkedIn 个人资料通常会显示在搜索结果的顶部，所以要确保你的第一印象是好的。

# 4.专注于客户服务

当你开始创业的时候，口碑对你事业的成功至关重要。你的早期客户应该向他们的朋友和同事称赞你，并推荐你。多做一点来表达你的感激之情，发送私人邮件，不断给他们惊喜。

# 5.你不能再回避公开演讲了

有效沟通的能力对于激励员工、获得客户以及向投资者推销以筹集资金至关重要。大多数人不太喜欢公开演讲。变得舒适。

# 6.选择合适的办公地点，但不要担心办公室

选择一个最适合你的业务需求的地点，一个提供发展机会、接近客户、接近供应商和适当竞争水平的地点。如果你没有任何收入，先不要担心仓库或办公室。

![](img/53f6ac55af76ee7087a304ceb07a0354.png)

# 7.这需要一个团队

你不能独自做这件事。投资者更愿意投资于团队而不是个人，因为伟大的团队才能创造伟大的产品，最终创造伟大的公司。此外，要经常招聘。在最初的日子里，你会经历低保留率。不要等到有需求时才开始招聘。你可能最终得到低于标准的候选人，这可能比人手不足对你的业务造成更大的伤害。

# 8.继续

主动带来魔力。保持能量流动，保持创意，保持推销，保持接触潜在客户，伟大的事情最终会发生。当小胜利真的发生时，尽情享受那些时刻。宇宙在轻拍你的背。停止分析一切。是癌症。不断尝试新想法，快速学习。

# 9.你没有时间追求完美

速度永远胜过完美。好到可以出货。不建设，就不学习。美妙之处在于，首先你会很快意识到你的假设有多么错误。你的事业正在进行中。现在，如果你快速推出你的产品或服务，你可以建立一个客户社区，提供有价值的反馈，帮助你改进产品。

# 10.适应性强

计划变了。开始时没有计划是愚蠢的，但是坚持一个被误导的计划更糟糕。当你的直觉告诉你的时候，坚持你的决定，但是作为一个企业家，你必须敏捷和开放。取得适当的平衡。

# 11.搭档

与你所在领域的其他创业公司携手，建立一个互补的技术，是一个合理的战略，几乎会立即为你的产品增加很多价值。这是一个有效和廉价的举措，帮助您更快地打入市场，在这个过程中学习，并加快您的产品路线图。

# 12.重要的时候亲自出现

像 Slack 和 Skype 这样的虚拟工具使我们能够与世界上任何地方的任何人合作。然而，当事情悬而未决时，没有什么能比得上一次面对面的会面。那是你真正团结起来，让事情发生的时候。此外，永远不要拒绝与新朋友交流的邀请。意外之喜是真实的，它能为你的生意创造奇迹。

# 13.把你的故事联系起来

品牌是关键。从一开始就把你的创业公司看作一个品牌。有意识地设计你的创业公司与外部利益相关者的每一次互动。密切关注你的名字、网址、网站设计、内容创作、语气、招聘启事、电子邮件签名和其他细节。这将巩固你的品牌，并在向潜在客户推销或招募新团队成员时产生巨大影响。不要忘记你作为创始人的品牌。你的网上形象应该是一流的。一天下来，谷歌搜索对你在数字世界中的第一印象起着巨大的作用。

# 14.实际一点

可以快速烧钱。作为一个白手起家的企业家，你不能犯这样的错误。避免价格过高的会议、办公室开销和其他启动费用，它们会很快增加。在尽可能长的时间内保持事情尽可能的精简。它会让你维持足够长的时间，找到适合市场的产品，并建立一个伟大的公司，否则它就会死去。一些初创公司资金雄厚，却没有足够的实力来创造收入。他们习惯了这样的想法，即烧钱是正常的，因为他们有一个受资助的公司。他们关注的是筹集下一轮资金，而不是建立一个有利可图的业务。高度的实用性是成功的关键。

# 15.创造价值

在当今世界，在所有的噪音中被听到比以往任何时候都难，获得人们分散的和来之不易的注意力的关键是创造价值。花时间创造性地思考如何以一种有用甚至有趣的方式展示你的解决方案的价值。这可能意味着设计一个有新闻价值的噱头，比如跑马拉松来展示你的无汗商务衬衫产品，甚至是制作一份精心策划的时事通讯。一旦你专注于创造价值，你就会停止追逐线索。

# 16.保持谦逊

自信和自尊是展现韧性的关键。但是，如果你让你的自我妨碍了你，你就不会准备好采取必要的步骤来获得成功。这可能意味着做一些琐碎的工作，比如发送冷冰冰的电子邮件，或者向更有经验的企业家寻求建议，甚至倾听你的客户质疑你的假设。大多数情况下，你不会处于强势地位，所以尽早接受它。拥抱你的脆弱。它会让你变得更强。不要害怕测试一个产品，即使它不像你想象的那样完美，并且你对此感到有点羞愧。你最终会从你的错误中学习，并在职业和个人方面成倍增长。这是旅程值得的原因。

# 17.这是一次孤独的旅程

它会渗透到你的个人生活中。你一开始只会谈论生意。你的朋友和家人最终会厌倦的。没有人会真正理解。你是一个为生存和发展而奋斗的小企业主。你统治世界的伟大计划在支付账单的过程中迷失了。尤其是在第一年，你所有的时间和精力都将用于建立你的企业。这会占用你和家人在一起的时间。这是一个巨大的牺牲，你必须意识到所需要的承诺程度。

# 18.照顾好你自己

开始时，你会被工作消耗殆尽；如果你休息一天，你会吃得很差，很少锻炼，并且会有负罪感。休息一下不仅对你的身心有好处，对你的事业也至关重要。偶尔后退一步是必要的。这将有助于你在前进的道路上更加清晰，并有助于激发你的创造力。

# 19.你会是最后一个拿到钱的人。

作为首席执行官，你是最后收支票的人。这就是它的运作方式，直到有足够的收入。

# 额外提示:

大多数已经存在一段时间的企业家都经历过成功和失败。这两种情况都给你上了宝贵的一课，告诉你什么能让你成功，什么会导致失败。当你开始创业的时候，你不需要想好所有的事情，但是要确保你是出于正确的原因开始创业的。我们生活在一个大多数人都怀有创业雄心的时代，市场上已经出现了创纪录数量的初创公司。不是所有的想法都行得通；很多人会因为这个时代而在经济上和情感上受到伤害。

尽管如此，我还是祝你在新的旅程中好运。世界需要企业家。这是为自己和他人创造真正价值的最佳途径。不管怎么样，我怀疑你会后悔搬家。
