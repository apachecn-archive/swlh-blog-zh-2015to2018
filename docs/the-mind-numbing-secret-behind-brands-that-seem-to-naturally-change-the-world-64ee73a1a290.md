# 改变你的故事，改变世界

许多公司在开发或改进产品时，关注其品牌的特点、独特之处以及它们提供的价值。都是很正当的活动。

但是当涉及到告诉世界的时候，他们忘记了 ***这是真正的工作开始的地方*** 在这里你需要和开发你的产品时一样多的创造力。

你必须投入两倍、三倍、5X 的创造力，才能获得你应得的吸引力。

# 如何改变世界(以及如何停止为自己的命运找借口)
