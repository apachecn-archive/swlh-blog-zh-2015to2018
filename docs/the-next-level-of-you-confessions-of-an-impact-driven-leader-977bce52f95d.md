# 下一个层次的你:影响力驱动型领导者的自白

> 原文：<https://medium.com/swlh/the-next-level-of-you-confessions-of-an-impact-driven-leader-977bce52f95d>

## 让你成功的品质不会让你成功

![](img/8974872e97a7c5c15a1dce6e208463ac.png)

你被驱使。你有影响力。你做的每件事都很出色。你已经完成了伟大的事业。你受到同事和朋友的喜爱和欣赏。从表面上看，你是成功的。

但是最近你开始意识到你渴望更多的东西。你错过了一些东西，但你不太确定是什么。你有一种感觉，你把自己的很大一部分留在了桌子上，不知何故，你可以*变得更*。

你的驱动力、你的影响力、你的完美主义和你的可爱已经让你走得很远了。**但是正是这些品质阻碍了你获得更大的成功和成就。**

要成为下一个层次的你，你需要重新思考成为你意味着什么。

是时候蜕皮了。是时候[升](http://investorfieldguide.com/growth-without-goals/)级了。

# 动力和完美

你雄心勃勃、有动力且独立。你多才多艺。你总是有解决问题的诀窍。你一生都表现出色，超出预期。既然你能应对任何挑战，你总是能做得更多。因为其他人只会拖你的后腿，所以你更喜欢自己处理事情。你为更大更广的挑战而奋斗，因为你知道你能应付它们。

但是你已经开始问自己:我是如何被*推向*的？我被驱动*是什么？这么忙，我要去哪里？*

你已经意识到你实际上并没有被*推向*任何东西。你唯一被驱使的是你肩上那个无情的、不停的奴隶主，他鞭策你做更多、更多、更多。但这对他来说永远都不够，是吗？如果有什么不同的话，那就是你被驱离了自己和你想要的。事实上，你已经不知道你真正想要的是什么了，因为它已经不重要了。再加上你高标准的成就，你就被困在了一个你不想克服的无限高的障碍中。高标准让你看不到自己真正想要的东西。因为你努力追求完美，所以无所事事比无法克服高标准更容易。你不寻求帮助。所以现在你被困住了。你已经麻木了。

你的干劲让你受益匪浅。但这也让你停滞不前，麻木了。

什么让你在这里，不会让你在那里。

# 影响和价值

你专注于传递影响力和价值。你不喜欢乱搞。你想对世界产生真正的影响，所以你鄙视浪费时间。如果有价值，你就行动。如果没有，你就像躲避瘟疫一样躲避它。你一生都把价值放在首位。你总是认为玩耍是给孩子们的。你可以被信赖去完成事情。你做到了。

但是你对高效的关注意味着任何没有价值的、实际的、可衡量的最终目标的事情都不值得去做。休息，恢复，休息，玩耍，只是闲逛——何必呢？你会吃奇亚籽、姜黄和椰子油，直到它们从你的耳朵里溢出来，但一个古老的巧克力蛋糕呢？是的没错。毕竟，玩耍和快乐是孩子们的事。因为你无情的价值取向，你变得越来越孤立。你太专注于富有成效和做“正确”的事情，以至于忘记了那些被称为快乐和喜悦的情绪是什么感觉。你变得如此专注于*做*，以至于你害怕*成为*。

你对价值的关注让你受益匪浅。但它也阻碍了你实现真正的、真实的、深层次的影响和联系。

什么让你在这里，不会让你在那里。

# 善良和同情

你很可爱。你很有礼貌。你真好。人们喜欢你的陪伴。你已经学会了用你微调过的情商来玩社交游戏。你可以融入任何团体，无论是在工作场所还是在酒吧。你是最高级别的外交官，总是看到事情的另一面。每个人都喜欢你。

但你不会潜得很深。你害怕承诺。你害怕脆弱和示弱。你回避冲突，因为你把争论的不适与你作为一个人的价值联系在一起。保持和平变得如此重要，以至于你已经学会忽视自己的意见。你已经变得如此习惯于让别人快乐，以至于你忘记了什么是真正的快乐。

你的讨人喜欢让你受益匪浅。但这也阻止了你体验爱的温柔和实现自己的愿望。

什么让你在这里，不会让你在那里。

# 工作

事情变成这样是完全自然的。你已经取得了这么大的成就。但是现在你可以开始看到代价了。你不仅剥夺了自己的活力之美。你正在剥夺*这个世界*你自己和你潜在的影响力。在你努力追求影响力、富有成效和善良的过程中，你最终陷入了困境、麻木和孤立。在试图把别人放在第一位的时候，你实际上已经优先考虑了自己的不安全感和防御机制。

那么，该怎么办呢？首先——承认所有的优点。欣赏并感谢你所有令人惊叹的品质。他们带你走了很远。不要试图“修理你”。没什么不好的。

但是——这是一个很大的、令人毛骨悚然的*但是*——要迈出下一大步，实现你真正想要的，你需要融入你的核心品质的一个更微妙的版本。你需要了解它们从哪里来，它们通向哪里。你需要开始看到你目前在补偿什么，并开始治愈你灵魂深处的创伤。你需要扩展成为你的意义，不管这看起来有多不舒服。

从哪里开始？以同样的方式学习任何东西。*练习*。练习做那些让你不舒服的事情和你目前正在抵制的事情。

你可以练习开放。带着*被*的不适感练习坐姿。练习注意你想要什么，然后去追求它。练习承诺你的愿望。练习打开你的心扉，把它留在地板上，让所有人看到它的凌乱和血腥。上帝保佑，练习允许自己吃那块甜美的巧克力蛋糕，并且真正地享受它。

和你的教练一起努力。向你的密友和同事寻求反馈。通过深入的内在工作向他人展示你的领导力。如果你敢的话，尝试一下你的发展。

成为下一级的你并不容易。没有快速解决的办法。有时会非常讨厌。这就像蛇蜕皮一样——一点也不优雅。就像蛇一样，这不是一次性的。这是一生的工作。

我亲爱的朋友，这才是真正的工作。这就是*真正影响力*的来源。你已经意识到，这是你的下一个大项目。是*你*。

但你一直喜欢挑战，对吧？
