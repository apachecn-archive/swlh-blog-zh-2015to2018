在罗马帝国时期，一种常见的关系是赞助人与他的天线。通常是艺术家、作家或表演者，这种关系主要由完成小任务、传递信息或跑腿以换取金钱、食物或恩惠的 Anteambulo 组成。

Anteambulo 的字面定义是“一个走在前面扫清道路的人”，事实上，这是一个年轻艺术家要完成的任务…
