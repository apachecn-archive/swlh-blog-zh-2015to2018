# 为什么超专业会让你立刻成为顶级自由职业者

> 原文：<https://medium.com/swlh/why-being-ultra-professional-instantly-makes-you-a-top-tier-freelancer-11e64860f27e>

## 比精通你的手艺或多年的经验更重要的是，成为一个值得信赖的职业选手能让你在竞争中领先一大步。

![](img/70eb23c65a9d5fb6627a20514d677605.png)

这是我反复强调的一个主题，但怎么说都不够。

传统上，自由职业者有一个污名:一种刻板印象，我们都是古怪的，不可靠的，不善于沟通的人，不能被依赖。

如果你是像我一样的设计师——或者其他“有创造力”的类型——那就更糟了。创意人员有他们自己的难以共事的刻板印象，他们是“疯狂的天才”，喜欢独自工作，通过神秘的过程，并且在追求他们的手艺时经常不能满足严重的业务期限。

结合自由职业者和设计师的假定失败，你看起来就像一个非常悲惨的标本。

# 我们是怎么到这里的？

大多数数字技能的准入门槛几乎为零，因此市场上自然充斥着缺乏经验、不可靠的“牛仔”类型的人，他们认为这项工作是轻松成功的敲门砖。世界各地的企业都与不专业的自由职业者和设计师打过交道，这些重复的不合标准的经历在我们潜在客户的集体嘴里留下了酸味。

然而，与自由职业者合作的吸引力仍然存在:通常比他们的代理同行成本更低，更直接和透明的关系，没有管理层，中间人或加价。

所以他们一直在寻找我们，每次都希望下一个会更可靠一点。这就是你和我的切入点。

当他们找到专业人士时，就像呼吸了一口新鲜空气。他们觉得自己中了头彩:与独立顾问合作的好处，加上他们期望从一家大型知名机构获得的可靠业务关系。

你可能不如他们过去共事过的其他人有才华，或者你受教育的年限或经验更少——但这些都比不上他们从与一个他们可以信任和依赖的承包商的轻松关系中感受到的快乐。

这是字面上所有需要的。你的技能和经验能让你参与进来，但是专业能赢得交易，建立长久的[合作关系。](/swlh/professional-freelancers-dont-do-gigs-1312d4fee496)

## 真的那么容易吗？

是啊！

市场上仍然挤满了牛仔，而且一天比一天拥挤。[底层食者不断繁殖并互相吞食](/swlh/freelancing-upwork-ea8205aadd1)，但你所要做的就是把你的头抬高到它们之上，让它们变得无关紧要。

磨练自己的技能。掌握你的手艺。把这些都做了。但如果你只能做一件事，那就练习超专业。这比其他任何事情都更能让你难忘。

# 什么叫超专业？

这并不意味着你必须穿西装打领带去见客户。这不是看起来像个角色(虽然那也有帮助！)是关于*成为* it。

*   **靠得住。承诺你最大的努力，并超额完成。每天准时出现(你是自己的老板，并不意味着你应该懈怠。让你和你的客户都失望)。不要让任何东西从缝隙中掉出来。成为你的客户想要[依赖的人，长期照顾他们的商业利益](/swlh/are-you-working-for-cost-clients-or-value-clients-bc5a4f517ef2)。**
*   **沟通。** *结束*交流！尽早并经常设定期望值。描述你的过程。让你的客户参与进来。激发良好的反馈。有求必应。发送每日更新，知道如何写好简洁的邮件。对任何事情都要诚实和透明。最重要的是，知道如何用你的客户能理解的语言交谈，不要使用技术/设计术语。
*   **待人和善，尊重他人。避免成为一个混蛋。展示你想要的职业关系类型，你会得到更多同样的回报。**
*   **变得有条理。**了解并信任你的流程，当你学习到更好的方法时再去改进。做好你的家庭作业，有备而来。掌握这一过程，并引导你的客户完成这一过程。[精确管理您的项目和进度](/swlh/smoothing-out-the-freelance-rollercoaster-4600a5b3a77c)。尊重你的客户和你自己的时间。
*   提供卓越的价值。超越像素去思考，寻找意想不到的方式来提供与你的创造性工作相关的商业策略。你对客户的业务越有激情，你就越能全心全意地让他们成功。你的客户会注意到这些额外的努力。
*   为每一个客户和每一个项目做好工作。以高标准要求自己，永远不要放松。你会建立起一个价值连城的好名声。

# 上升到底部进食者之上

奶油升到顶部，或者他们是这么说的。做那个奶油。

保持超专业的名声会让你比其他任何事情都要快。虽然自由职业者没有成功的秘诀，但这是你能找到的最接近的捷径。

如果你厌倦了与只为几便士而工作的人竞争，或者与不重视或不尊重你的专业知识的客户斗争——超越那种竞争性的底层喂养，通过**击败他们成为最专业的**。你会在不知不觉中赢得自由职业的比赛。
